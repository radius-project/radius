## Radius v0.57.1

This patch release pins the Deployment Engine image installed by the Radius Helm chart to the `0.56` tag to work around a bug in Deployment Engine `0.57.0`, and pins the Bicep CLI used by the `rad` CLI and CI workflows to `v0.42.1` to work around a breaking change in Bicep `v0.43+` that rejects local registry targets (e.g. `br:localhost:5000/...`). Check out the [changelog](#changelog) for more details of what was addressed in this patch.

## Changelog

* Pin Deployment Engine image to 0.56 for v0.57.1 patch release by @brooke-hamilton in https://github.com/radius-project/radius/pull/11855
* Pin Bicep CLI to v0.42.1 in workflows and rad CLI by @sk593 in https://github.com/radius-project/radius/pull/11812
